   Installation & Configuration
 Download
Rename your downloaded file to webbreaker or for Windows webbreaker.exe

 Environment Settings

Add it to your environment's $PATH or for Windows %PATH%

 Install Configuration

First time installs, run the webbreaker command in your terminal to create the default configurations, under $HOME/.webbreaker/config.ini

 Post-Configuration

Configure the Webinspect, Fortify SSC, and ThreadFix sections to your environment. See User Guide for advanced set-up.
# Change to your Fortify SSC URL and run `webbreaker admin credentials --fortify`
[fortify]
ssc_url = https://ssc.example.com/ssc
username =
password =

# Change to your ThreadFix URL and create a ThreadFix API Key
[threadfix]
host = https://threadfix.example.com:8443/threadfix
api_key = ENTER API KEY HERE

# Change to your WebInspect server(s)
server_01 = https://webinspect1.example.com:8083
endpoint_01 = %(server_01)s|%(large_server_max_concurrent_scans)s

That's it, you're done! Happy scanning!
